# Online Shop

***

Development Online Shop app with CRUD operations:
* application Login
* application Logout
* adding request
* update request
* remove request
* filter requests
* pages of Help and Message
* error processing and loaders loading

***

```
Technologies and libraries in development:
- Vue 3
- Vuex
- vue-router
```
